## 3 开发

### 3.1 安装依赖

```
yarn install
```

### 3.2 本地开发

```
yarn serve
```

### 3.3 构建

```
yarn build --report
```

### 3.4 Lints and fixes files

```
yarn lint
```

### 3.5 预览

```
yarn preview
```
